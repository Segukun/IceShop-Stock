# IceStock

IceStock es una aplicación web local diseñada para gestionar el stock de una heladería de forma simple y eficiente. Permite a los empleados registrar productos, controlar cantidades y llevar un seguimiento de los movimientos de stock.

## Funcionalidades
  - Crear, editar y eliminar productos
  - Visualizar el stock actual
  - Registrar entradas (IN) y salidas (OUT) de stock
  - Consultar el historial de movimientos
  - Detectar productos con bajo stock
  - Buscar y filtrar productos por categoría
  - Tecnologías
  - Frontend: React
  - Backend: Node.js + Express
  - Base de datos: MySQL / SQLite


## Uso
El sistema está pensado para ejecutarse de forma local dentro del negocio.
No requiere usuarios ni autenticación, ya que su uso es interno por parte de los empleados encargados del stock.
